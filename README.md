# Mimeng Style Writing Skill

This repository contains a Codex Skill for Chinese self-media writing workflows.

## What It Includes

- `SKILL.md`: main skill workflow and usage rules
- `references/`: eight full-review methodology files
- `agents/openai.yaml`: agent configuration

## Main Capabilities

- Topic selection
- Title generation
- Material and quote discovery
- Article structure planning
- Full article drafting
- Draft revision and polishing
- Xiaohongshu note drafting

## Usage

Install or copy this folder into a Codex skills directory, then ask for tasks such as:

```text
按这个 Skill，围绕「中年失业不可怕，勇敢踏出第一步」写一篇公众号文章。
```

For full article tasks, the Skill defaults to an eight-step workflow:

1. Topic judgment
2. Core viewpoint
3. Title
4. Materials
5. Article structure
6. Draft
7. Quotes
8. Final polish and publishing checklist

## Notes

This Skill is designed to learn writing methods, structure, and workflow patterns. It should not be used to copy original articles or impersonate any real author.
